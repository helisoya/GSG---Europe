# GSG - Europe

<img class="img-fluid" src="https://helisoya.github.io/Portfolio/assets/img/portfolio/GSG-E.png" alt="logo" height=350 width=600>

**GSG - Europe est un jeu de stratégie en 3D codé sur Unity. C'est une amélioration de GSG 2 dans le but de le publier sur itch.io.**

Le Joueur contrôle un pays d'Europe et doit faire face à ses voisins.

## Technologies utilisées
- Unity Engine
- C#

## Caractéristiques
Le jeu est un premier lieu une amélioration de son prédécesseur, GSG 2.

- Les temps de chargement pour lancer le jeu ont été réduis.
- Les guerres ont été refaites, il faut maintenant faire un traité de paix pour pouvoir annexer une province.
- Un effet de shader a été ajouté afin de mieux comprendre différencier les provinces et les provinces occupées.
- Une limite d'unités a été ajoutée afin d'éviter un surplus d'unités chez les IAs.
- Un système de priorités nationales a été ajoutés, permettant d'obtenir des buffs rendant sa nation plus forte.
